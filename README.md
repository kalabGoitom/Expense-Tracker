# Expense Tracker

A simple and user-friendly **Expense Tracker** built with **vanilla JavaScript**, HTML, and CSS.  
The app allows users to track income and expenses, view their balance in real time, and store data locally in the browser.

---

## Features
- Add income and expense transactions  
- Edit and delete transactions  
- Real-time balance calculation  
- Persistent data using `localStorage`  
- Dark mode / light mode toggle  
- Clean and responsive UI  

---

## Technologies Used
- HTML5  
- CSS3  
- Vanilla JavaScript  
- Browser Local Storage API  

---

## How It Works
- Users enter a description and amount  
- Select transaction type (income or expense)  
- Transactions are stored in `localStorage`  
- Balance, total income, and total expenses are updated automatically
